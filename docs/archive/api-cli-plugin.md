## plugin

Get interactive options for plugin

```bash
rnv plugin
```

### list

list all available plugins

```bash
rnv plugin list`
```

### add

Add new plugin to your project

```bash
rnv plugin add
```

### update

Update existing plugin

```bash
rnv plugin update
```

### help

Display plugin help

```bash
rnv plugin help
```
