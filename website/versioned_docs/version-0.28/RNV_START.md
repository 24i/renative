> start server / bundler on selected platform

## Task Order

🔥 `configure` ➡️ `start`  ✅

## start

Get interactive options for start

```bash
rnv start
```

### help

Display start help

```bash
rnv start help
```

## Options

`-p`, `--platform` - Specify platform

`-r`, `--reset` - Reset bundler cache

`--mono` - Monochromatic output to terminal (no colors)
